# Terrier Retrival

BatchRetrieve is one of the most commonly used PyTerrier objects.
It represents a retrieval transformation, in which queries are mapped to retrieved documents. 
BatchRetrieve uses a pre-existing Terrier index data structure, typically saved on disk.
